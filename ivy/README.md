Ivy
===

```sh
# Install JS deps
npm install
# Install and set up Postgres database
brew install postgresql
pg_ctl init
createdb
pg_ctl start
psql -c "CREATE DATABASE ivy_dev;"
psql -c "CREATE DATABASE ivy_test;"
make migrate

# Start test watcher
make test

# Compile the frontend
make frontend

# Run server
make server
```
